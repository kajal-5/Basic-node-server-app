# NodeJS URL Routing

A basic Node.js project demonstrating URL routing using the built-in `http` module.

# Features

- Create HTTP server using Node.js
- Handle multiple routes
- Send custom responses based on URL
- Return Page Not Found for unknown routes

# Routes

| URL | Response |

| `/home` | Welcome home |
| `/about` | Welcome to About Us |
| `/node` | Welcome to my Node Js project |
| Any other route | Page Not Found |

# Run Project

Install Node.js, then run:

```bash
node app.js
